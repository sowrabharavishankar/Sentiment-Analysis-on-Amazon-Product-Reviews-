# Amazon Reviews Sentiment Analysis

This project performs **Sentiment Analysis** on Amazon product reviews to classify them into **positive**, **neutral**, or **negative** sentiments.  
Using **Natural Language Processing (NLP)** and **Machine Learning**, the notebook cleans, analyzes, and visualizes customer opinions to uncover patterns and insights from textual feedback.

---

## Project Overview

Customer reviews for understanding user satisfaction and product performance.  
This project:
- Cleans and preprocesses raw Amazon review data  
- Computes sentiment scores using NLP  
- Classifies reviews into **Positive**, **Negative**, and **Neutral**  
- Visualizes sentiment distribution, trends, and word frequencies  

---

## Features

- **Data Cleaning:** Removes irrelevant columns, missing values, and noise  
- **Sentiment Scoring:** Uses NLP tools (e.g., VADER/TextBlob) to calculate sentiment polarity  
- **Categorization:** Converts sentiment scores into labeled classes  
- **Visualization:** Includes multiple plots:
  - Sentiment distribution  
  - Average rating by sentiment  
  - Sentiment trends over time  
  - Word clouds for each sentiment  
  - Pie chart of sentiment proportions  
  - Review length vs. sentiment  

---

## Tech Stack

- **Language:** Python  
- **Libraries:**  
  - `pandas` — data manipulation  
  - `matplotlib`, `seaborn` — visualizations  
  - `nltk`, `textblob`, `vaderSentiment` — NLP and sentiment analysis  
  - `wordcloud` — text visualization
